      # Getting Started

      # Run project

      # clone
git clone git@github.com:JeremyWu917/jeremy-press.git

# change dir
cd jeremy-press

# install dependencies
yarn

# run dev
yarn dev

      TIP

      You can enjoin the home page at url http://localhost:8080/jeremy-press/ ☕️
